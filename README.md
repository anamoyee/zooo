# How to use
1. Install [uv](https://docs.astral.sh/uv/getting-started/installation/)
2. Clone this repo
3. Run `uv sync`
4. Create `ids.txt` file, a list of newline-separated discord IDs (or profile IDs) of the profiles you wish to fetch.
5. Run `zoo`/`zoo.exe`
    - (on windows) `.\.venv\Scripts\zoo.exe`
    - (on linux) `./.venv/bin/zoo`

## Run `zoo --help` for extra help
